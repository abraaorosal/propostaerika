# propostaerika

Landing page comercial criada para apresentar a proposta do Ecossistema Digital de Vendas e Gestao do Projeto Evento Outubro.

## Estrutura

- `index.html`: pagina principal
- `styles.css`: identidade visual e responsividade
- `script.js`: animacoes e interacoes leves
- `desenrolaai.jpg`: logo da marca

## Publicacao

Este repositorio esta preparado para publicar automaticamente no GitHub Pages via GitHub Actions a cada push na branch `main`.
